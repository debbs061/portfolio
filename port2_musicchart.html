사이트별 음악 차트 크롤링 (Musicchart Crawling)

음악 차트 사이트 FLO 와 ITUNES 의 Top 100 데이터를 매일 새벽 4시마다 크롤링 해오는 프로그램 입니다.
  • 날짜를 파라미터값으로 받아 url 요청에 맞게 음악사이트 FLO와 ITUNES 의 Top100 데이터를 JSON 형태로 뿌려줍니다.
  • FLO는 Open api 를 제공해서 api를 이용했습니다.
  • ITUNES는 제공되는 api 가 없어서 Top 100 페이지를 크롤링 했습니다.
  • 특정 키워드로 파라미터로 넘기면 가수별, 곡별, 앨범별 검색결과가 나옵니다.
  • 매일 새벽 4시마다 크롤링을 진행하도록 Spring Scheduler 를 이용했습니다.
