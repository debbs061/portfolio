레시피 월드 (Recipe World)

나만의 냉장고에 식재료를 등록하고, 원하는 레시피를 검색할 수 있는 콘솔 프로그램 입니다.
  • 로컬 폴더에 메뉴별로 텍스트 파일을 생성하여 회원별 데이터를 저장합니다. (텍스트 파일이 DB 역할을 합니다)
  • 바이트 기반 입력 스트림인 FileInputStream 클래스로 파일을 바이트 단위로 읽어들이고, ObjectInputStream 클래스를 통해 이 연속적인 바이트를 객체로 복원하는 역직렬화 작업을 수행합니다.
  • 파일 입출력 스트림을 이용하여 CRUD 작업을 구현했습니다.
  • 식재료 검색 시 네이버 최저가 랭킹순을 보여주기 위해 네이버 검색 API를 이용합니다.
  • GET으로 호출할 때 HTTP Header에 애플리케이션 등록 시 발급받은 Client ID와 Client Secret 값을 같이 전송해줍니다.
  • 레시피 자료는 농림축산식품 공공데이터 포털의 OpenAPI 를 이용했습니다.
  • Node 인터페이스를 이용하여 xml 파싱을 구현했습니다.
